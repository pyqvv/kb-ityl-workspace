# KB IT's Your Life Workspace 💻

KB IT's Your Life 과정에서 진행한 Java/JavaScript 학습 코드와 일일 실습을 모아 둔 저장소입니다.  
기초 문법부터 객체지향, 컬렉션/스트림, 프론트엔드 기초(Vue/Bootstrap)까지 단계적으로 학습할 수 있도록 구성되어 있습니다.

## 📌 저장소 개요

- Java 학습 영역은 `java-exercises`, `java-practice`로 분리
- JavaScript/Vue 학습 영역은 `js-callback`, `vuejs`, `kb-daily-practice`에 정리
- 대부분 실습형 코드(`Todo`, `Example`)로 구성되어 빠르게 실행/복습 가능

## 📁 디렉토리 안내

### `java-exercises`
문제 풀이 중심의 Java 실습 모음입니다.

- `This is Java` 확인 문제(`tisjava`)와 OOP/상속/다형성/예외/컬렉션/스트림 주제를 포함
- `Todo` 기반 파일을 직접 완성하는 형태의 과제가 많음
- 상세 문서: [`java-exercises/README.md`](./java-exercises/README.md)

### `java-practice`
챕터별 Java 기본/심화 예제를 정리한 폴더입니다.

- `ch06`~`ch17` 범위의 교재형 예제(`chXX.secYY.examZZ`) 구성
- 클래스, 상속, 인터페이스, 예외, 제네릭, 컬렉션, 람다, 스트림 학습 포함
- 상세 문서: [`java-practice/README.md`](./java-practice/README.md)

### `js-callback`
JavaScript 콜백/고차 함수 연습 폴더입니다.

- 공통 `repeat` 함수 기반으로 출력/가공 문제를 단계별로 연습
- 상세 문서: [`js-callback/README.md`](./js-callback/README.md)

### `kb-daily-practice`
날짜별 웹 프론트엔드 실습 기록 폴더입니다.

- Vue 기본 구조 및 데이터 바인딩
- 컴포넌트 구조와 상태 관리
- 슬롯, provide/inject 등 컴포넌트 간 데이터 전달
- Vue Router 기반 SPA 구현
- Bootstrap UI 실습

### `vuejs`
Vue 기초 문법 단위 실습 파일 모음입니다.

- 선언형 렌더링, 반응형 변수, 이벤트 리스너, 조건부 렌더링, 반복 리스너, 계산된 속성 예제 포함
- 상세 문서: [`vuejs/README.md`](./vuejs/README.md)
