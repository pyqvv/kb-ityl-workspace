> KB IT's Your Life 과정에서 진행한 Java & JavaScript 기반 학습 과제와 실습을 정리한 저장소입니다.

기초 문법부터 객체지향, 컬렉션/스트림, 프론트엔드(Vue/Bootstrap)까지  
단계적으로 학습 내용을 기록합니다.

## 🧭 Overview

- Java: `java-exercises`, `java-practice`
- JavaScript/Vue: `js-callback`, `vuejs`, `kb-daily-practice`
- 실습 중심 코드(`Todo`, `Example`)로 빠르게 실행 및 복습 가능

## 🗂 Directory Structure

### ☕ Java

#### `java-exercises`
문제 풀이 중심 Java 실습

- OOP / 상속 / 다형성 / 예외 / 컬렉션 / 스트림
- 『이것이 자바다』 확인 문제 포함  
- [README 보기](./java-exercises/README.md)


#### `java-practice`
KB IT's Your Life 과정에서 제공된 Java 기본/심화 실습

- 교재형 구조(`chXX.secYY.examZZ`) 구성
- 객체지향, 예외, 제네릭, 컬렉션, 람다, 스트림
- [README 보기](./java-practice/README.md)


### 🌐 JavaScript / Frontend

#### `js-callback`
콜백 함수 및 고차 함수 연습

- 공통 `repeat` 함수 기반 문제 풀이
- [README 보기](./js-callback/README.md)


#### `kb-daily-practice`
날짜별 프론트엔드 실습

- Vue 기초 및 컴포넌트 구조
- 상태 관리 및 데이터 전달
- Vue Router 기반 SPA
- Bootstrap UI


#### `vuejs`
Vue 기초 문법 단위 실습

- 렌더링, 반응성, 이벤트, 조건/반복, computed
- [README 보기](./vuejs/README.md)
