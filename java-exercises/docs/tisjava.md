# This is Java 1~5장 확인 문제

## 📌 개요
이 저장소는 『This is Java』 1~5장의 확인 문제를 풀이하고 정리한 코드입니다.  
각 장의 핵심 개념을 직접 코드로 구현하며 복습하는 것을 목표로 합니다.

---

## 🎯 학습 목표
- 자바 기초 문법 이해
- 변수, 연산자, 제어문, 배열 개념 정리
- 문제 풀이를 통한 실전 감각 향상
