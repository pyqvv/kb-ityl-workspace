# OOP 실전형 시험 대비 문제 세트

객체지향 프로그래밍(OOP) 핵심 개념을 복습하기 위한
실전형 문제 10개를 풀이한 Java 프로젝트입니다.

---

## 📌 개요

이 프로젝트는 생성자, 메서드, 캡슐화, 싱글톤 등
객체지향의 주요 개념을 문제 풀이를 통해 정리하는 것을 목표로 합니다.

---

## 🎯 학습 목표

* 객체와 클래스 구조 이해
* 생성자 및 `this` 키워드 활용
* 메서드 오버로딩과 반환값 처리
* `static`과 싱글톤 패턴 이해
* 캡슐화 및 객체 배열 활용

---

## 🧩 문제 구성

| 번호 | 내용                       |
| -- | ------------------------ |
| 01 | Student 객체 생성과 필드 출력     |
| 02 | Member 생성자와 `this` 활용    |
| 03 | MemberService 로그인 / 로그아웃 |
| 04 | Rectangle 반환값 메서드        |
| 05 | Calculator 메서드 오버로딩      |
| 06 | Printer `static` 메서드     |
| 07 | ShopService 싱글톤 패턴       |
| 08 | Account 캡슐화              |
| 09 | StudentManager 객체 배열     |
| 10 | BankApplication 종합 문제    |
