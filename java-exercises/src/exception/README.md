# 예외 처리 연습

## 📌 문제
사용자로부터 문자열을 입력받아서 그 문자열을 숫자로 변환하여 출력하는 프로그램입니다.

## ⚙️ 요구사항
1. `convert`라는 메서드를 정의합니다. 
   * 문자열을 전달받아서, 그 문자열을 숫자로 변환하여 리턴합니다. 
   * 전달된 문자가 `null`이거나 문자열의 길이가 0이면 `IllegalArgumentException`을 발생시킨 후 메소드를 호출한 쪽으로 예외를 던집니다.
2. `main` 메서드에서 `convert`를 호출하고 결과를 출력합니다.
3. 예외가 발생되면, "예외가 발생되었습니다. 문자열을 입력하지 않고 Enter키를 누르셨습니다."라고 출력합니다.

## 🔗 코드

- [리팩토링 풀이](ExceptionRefactored.java)

```
숫자로 변환할 문자열을 입력바랍니다.

변환된 숫자는 123 입니다.
```
[ 사용자가 입력하지 않고 Enter키를 누른 경우, 화면으로 출력되는 문자열입니다 ]
```

예외가 발생되었습니다. 문자열을 입력하지 않고 Enter키를 누르셨습니다.
```
