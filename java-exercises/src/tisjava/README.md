# This is Java 확인 문제

## 📌 개요
이 저장소는 『This is Java』 교재의 확인 문제를 풀이하고 정리한 코드입니다.    
기초 문법부터 객체지향 프로그래밍, 예외 처리까지 각 장의 핵심 개념을 직접 구현하며 학습 내용을 체계적으로 정리하는 것을 목표로 합니다.

---

## 🎯 학습 목표
- 자바 기본 문법(변수, 연산자, 제어문, 배열) 이해
- 클래스, 상속, 인터페이스 등 객체지향 개념 구조적으로 학습
- 익명 객체, 예외 처리 등 중급 개념 익히기
- 문제 풀이를 통해 개념을 코드로 구현하는 능력 강화
- 패키지 구조를 활용한 코드 조직화 및 설계 감각 기르기

---
