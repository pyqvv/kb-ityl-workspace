# 컬렉션 자료구조 연습

## 📌 개요

Java 컬렉션 프레임워크(List, Set, Map 등)의 기본 사용법과 자료구조별 특징을 이해하고,   
데이터 저장 및 처리 방식을 실습을 통해 익힌다.

---

## 🎯 학습 목표

- List, Set, Map의 구조와 특징 이해
- 데이터 추가, 수정, 삭제 및 조회 방법 학습
- Stack과 Queue의 동작 원리 이해
- 컬렉션을 활용한 데이터 관리 및 처리 능력 향상
- 다양한 자료구조를 상황에 맞게 선택하는 능력 습득

---
