# Java Exercises

Java 기초부터 객체지향, 예외 처리, 컬렉션, 스트림까지 단계적으로 연습한 문제 풀이 모음입니다.   
각 주제는 패키지별로 분리되어 있으며, 대부분 `Todo` 형태의 실습 파일을 직접 완성하는 방식으로 구성되어 있습니다.

## 🎯 학습 목표

- Java 기본 문법을 코드로 익히기
- 클래스 설계, 상속, 다형성 등 객체지향 개념 체화
- 예외 처리와 컬렉션/스트림을 활용한 데이터 처리 연습
- 패키지 단위로 문제를 분리해 구조적으로 코딩하는 습관 형성

## 📁 디렉토리 구조

`src` 하위의 주요 패키지는 다음과 같습니다.

- `tisjava`
  - 『This is Java』 교재 확인 문제 기반 실습
  - `ch01_intro` ~ `ch18_io` 중 일부 챕터를 단계별로 풀이
- `oop.basic`
  - TV, Book, Account, Student 등 OOP 기초 구현
- `oop.exam`
  - OOP 핵심 개념 점검용 10문제 세트 (`p01` ~ `p10`)
- `inheritance.drink`
  - `Drink`/`Alcohol` 상속 실습 및 리팩토링 버전
- `polymorphism`
  - `animal`, `coffeeshop`, `factory`, `shape`, `videoshop`, `wheeler` 문제
- `collection`
  - List/Set/Map/Stack/Queue 기반 컬렉션 실습 (`p01` ~ `p10`)
- `exception`
  - 문자열 숫자 변환 예제로 예외 처리 연습
- `stream.test`
  - Stream 기본, 내부 반복, 파이프라인, 필터링/매핑/정렬/집계 실습
