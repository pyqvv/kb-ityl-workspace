# 📚 Vue.js 기초 문법 실습

이 저장소는 Vue.js 3의 기초 문법과 **Composition API** 및 **Options API**의 차이점을 학습하기 위한 예제 코드를 포함하고 있습니다.

## 📂 프로젝트 구조 및 파일 설명

### 01. 선언적 렌더링 ([01-declarative-rendering.vue](01-declarative-rendering.vue))
* **핵심 개념**: 템플릿 문법(`{{ }}`)을 사용하여 데이터를 화면에 출력하는 방법.
* **학습 내용**: 
    * `setup()` 함수를 사용하는 Composition API 방식과 `data()` 옵션을 사용하는 Options API 방식의 차이 이해.
    * `setup()`에서 정의한 변수를 템플릿에 노출하기 위한 `return` 처리.

### 02. 반응형 변수 ([02-reactive-variable.vue](02-reactive-variable.vue))
* **핵심 개념**: 데이터 바인딩 방식(`v-bind` vs `v-model`)과 `ref`를 이용한 상태 관리.
* **학습 내용**:
    * **v-bind**: 값 → 화면으로만 전달되는 단방향 데이터 바인딩.
    * **v-model**: 입력과 변수 값이 동기화되는 양방향 데이터 바인딩.
    * `ref()`를 사용하여 값이 변경될 때 화면이 자동으로 업데이트되는 반응형 데이터 생성.

### 03. 이벤트 리스너 ([03-event-listener.vue](03-event-listener.vue))
* **핵심 개념**: 사용자 이벤트(클릭 등)를 감지하고 처리하는 방법.
* **학습 내용**:
    * `v-on:click` 또는 `@click` 축약형을 사용한 이벤트 연결.
    * `ref`로 선언된 데이터의 실제 값에 접근할 때 사용하는 `.value` 속성 이해.
    * 함수를 통한 로직 분리와 이벤트 전파 방지(`preventDefault()`).

### 04. 조건부 렌더링 ([04-v-if.vue](04-v-if.vue))
* **핵심 개념**: 특정 조건에 따라 요소를 화면에 표시하거나 제거하는 방법.
* **학습 내용**:
    * `v-if`와 `v-else`를 활용한 조건별 UI 분기 처리.
    * 반응형 변수(`counter`)의 값 변화에 따른 실시간 DOM 업데이트 확인.

### 05. 반복 리스너 ([05-v-for.vue](05-v-for.vue))
* **핵심 개념**: 데이터 배열이나 객체를 순회하며 동일한 구조의 요소를 반복 렌더링.
* **학습 내용**:
    * `v-for="item in items"` 문법을 사용한 목록 생성.
    * `reactive()`를 사용하여 객체 형태의 데이터를 반응형으로 관리하는 방법.

### 06. 계산된 속성 ([06-computed-value.vue](06-computed-value.vue))
* **핵심 개념**: 기존 데이터를 가공하여 새로운 값을 도출하는 `computed` 속성.
* **학습 내용**:
    * **computed**: 종속된 데이터가 변경될 때만 재계산되어 성능을 최적화함.
    * `filter()` 함수를 활용하여 원본 배열에서 특정 조건을 만족하는 아이템만 추출하는 실습.

---

## 💡 주요 학습 포인트
1.  **API 스타일 비교**: 동일한 기능을 **Composition API**(함수 중심)와 **Options API**(구조 중심)로 각각 구현하며 차이점을 익힙니다.
2.  **반응형 시스템**: `ref`와 `reactive`를 사용하여 Vue가 데이터의 변화를 어떻게 감지하고 화면을 갱신(Proxy 객체 활용)하는지 학습합니다.
3.  **지시어(Directives)**: `v-bind`, `v-model`, `v-if`, `v-for`, `v-on` 등 Vue 전용 HTML 속성의 활용법을 습득합니다.
